O comando "window.prompt" retorna o valor como uma string. Para cálculos números, é preciso converter essa string em number.

Para converter números:

// Number.parseInt(n) = Retorna valores de números inteiros
// Number.parseFloat(n) = retorna valores com números reais, ou com vírgulas.

A conversão do comando window.prompt, de string para number, é realizada quando o comando é inserido dentro de parênteses no comando Number.parseInt ou parseFloat.

Exemplo:

var n1 = Number.parseInt(window.prompt("Digite um número"))

É possível utilizar apenas o comando "parseInt" ou "parseFloat"


As versões mais novas do JS permitem que se utilize apenas o comando "Number(n)" para a conversão.
Nesse caso, o próprio comando vai definir se o número é inteiro ou real.

Exemplo

var n2 = Number(window.prompt("Digite outro número"))
