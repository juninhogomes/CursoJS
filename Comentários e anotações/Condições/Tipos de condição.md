# Tipos de condição

* Condição Simples;

## Condição Simples

A `condição simples` ocorre quando há apenas **uma condição/hipótese verdadeira**.

```
if (condição) {
    [true]
}
```

Ela continua simples, mesmo que tenha uma operação para a condição falsa.

```
if (condição) {
    [true]
} else {
    [false]
}
```
