Variável é onde o programa guarda os dados. Uma especie de apelido,
ou atalho para algo que vai ser utilizado posteriormente.


// ' var nome = window.prompt("Qual é o seu nome?") ' // 

Esse comando diz que a variável (VAR) "nome" está recebendo 
(=) o código window.prompt. 

--- 

O comando "window" abre uma espécie de alerta na tela. 

Para exibir o resultado do comando, basta inserir a variável, 
no caso "nome", dentro de outro comando JS. 

Por exemplo: 

// ' window.alert("É um grande prazer em te conhecer " + nome); ' //

---

!!! O sinal de "+" fora das aspas junta, ou concatena, a 
variável com o restante do comando. !!! 

Caso queira colocar mais coisas depois da variável, não esquecer 
de concatenar novamente com o sinal de "+" 

Exemplo: 

// window.alert("É um grande prazer em te conhecer " + nome + "!"); //


---

Para criar uma frase utilizando as variáveis sem precisar concatenar, 
pode-se utilizar a frase dentro de acentos graves (crase) e as variáveis
dentro do comando  ${} (chaves).

Exemplo:

// windows.alert(`Uma frase que contenha ${var1} e também ${var2}`)
